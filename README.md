# Starkane

## Configurar ambiente local

Terminal 1

```bash
make katana
```

Terminal 2

```bash
make setup
```

Copiamos el World Address y lo pasamos como parametro

```bash
make torii <WORLD_ADDRESS>
```

[Playground de graphql](http://localhost:8080/graphql)

### Ya esta el ambiente configurado!
